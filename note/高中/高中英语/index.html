    高中英语¶

    比较级最高级¶

    单音节词/少数多音节词：

    • 基本变化：-er, -est
    • e结尾：加-r -st
    • 辅音字母+y结尾：去y加-ier, -iest
    • 双写辅音字母，再加-er, -est

    多数多音节词：词前加more, most

    特例
    good/well better best
    many/much more most
    far farther/further farthest/furthest
    bad/ill worse worst
    little less least
    old older/elder oldest/eldest

    时态(核心：动作时间)¶

    8种基本时态¶

    8种状态\时间 过去 现在 将来
    一般 did
    was/were done    		 do/does
    am/is/are done    		 will do
    will be done
    进行 was/ware doing
    was/were being done    		 am/is/are doing
    am/is/are being done    		 will be doing
    完成 had done
    had been done    		 have/has done
    have/has been done    		 ×

    时态重点：3个过去时和现在完成时

    两种特殊时态¶

    过去将来时（在过去谈论将来，需要2个时间概念体现）：would do

    完成进行时（强调过去到现在一直在发生，表示最近一直）：have been doing

    虚拟语气¶

    主句 从句
    过去 would have done had done
    现在/将来 would do did

    标志：if/would have done/otherwise/or/without/but for…

    需特殊处理的：wish/as if/as though+

    • 过去：had done
    • 现在：did
    • 将来：would do

    区分：

    • would：主句一定有，从句没有
    • have：过去一定有，现在将来一定没有
    • be动词没有was只有were
    • 根据空格当前句子时间判断
    • 绝对没有will

    done：强调被动或已经完成

    阅读理解¶

    阅读理解潜规则：

    • 顺序对应原则：题目顺序一般就是答案在文中的顺序
    • 同义替换
    • 关键词原则

    文章重点：

    • 段首段尾
    • but等转折后是重点（while/though等引导的逗号之前的不是重点）
    • 比较级和最高级：划圈

    方法：

    1. 先读题目，后看文章
    2. 只关注有用信息，不要让无效信息浪费时间
    3. 注意隐含义，勿纠结字面意思。可借助近义词/反义词
    4. 切勿凭感觉做题

    关键点¶

    重选项，轻文章：

    • 查陌生单词对解题没任何帮助，换个文章，工作全白费
    • 主旨最重要，问主旨/目的/希望，不应该是赚钱/打广告（比如promote a product），高度不够
    • 文章结构一般都是总-分-总，中间的段落一般是论点/例子，能少读就少读
    解决问题 需要
    从文章中学到什么/文章标题 文章主旨 第一（和二）/末段，中间的段落都是次要信息
    细节题/某段重点/某句重点 段落主旨 but等转折词后是重点，看语境正负态度
    划线单词意思/段落大意等题（最难的问题） 看上下文意思，语境，正负态度

    • 主旨题，一定和文章主旨强相关，可先排除完全无关选项

    • 不要由繁到简：比如truthful→truth
    • 要按要求

    考点：

    • 有提示词：
    • 动词：时态语态，非谓语，词形变换
    • 名词代词：单复数，人称变化，反身代词
    • 形容词副词：词形变换，比较级最高级
    • 无提示词：
    • 连词关系词：主从句，逻辑关系
    • 介词
    • 冠词
    • 特殊选择：and/or/but

    无提示词：空格只填1个词¶

    如果空格后是

    • 冠词：a/an/the
    • 介词：介词短语积累，与on/in/at/with/about/for/by/of/over/under/to等的固定搭配
    • and/or/but
    • 名词短语：

    • 从句关系词：主从句划分和完整性判断

    • 连词

    • and/or/but

    句子和短语的根本句别：是否有谓语/时态（被动肯定是句子）

    关系词：what which who where how whether why when

    连词：因果，转折，并列，条件

    解法：

    主句完整？ 从句完整？ 用词
    × × what 物
    who 人
    × √ when where why how whether
    that：不缺意思
    √ × that/which 物
    that/who 人
    whose 谁的
    √ √ when 时间
    where 非时间
    （why, that)

    主句完整，从句不完整，注意：

    • 逗号后不能用that，用who/which
    • 介词后不能用that，用whom/which
    • 不定代词后只用that，如something that, all that
    • 最高级/极端概念后只用that，如best/first/-est that

    _ culture is rich in…

    从句不完整：缺少限定词，要用whose

    完形填空¶

    考察：

    • 词汇：词义，辨析，搭配
    • 上下文：
    • 瞻前顾后：通览全文，重视首尾句，预测主题
    • 左顾右盼：逻辑，上下文
    • 顾全大局：符合答案，全面检查
    • 义无反顾：除非逻辑和小词辨析错误，尽量不大面积修改
    • 短语和搭配：动词，名词和代词是重点；形容词，副词，介词，连词相对较少。

    做题要点：

    • 100%拿下第一句：找主线，拆首句
    • 不把时间花在“扫描”上，而是“锁定”答案上
    • 随时能返回来做题：往往空后线索占大部分

    关键：不要瞎蒙，证据不足就是蒙

    复习建议：

    • 找主线，拆首句
    • 搜集阅读材料：励志/温情，先抑后扬的故事
    • 突破高频词汇、短语和搭配，词义辨析，形容词副词，介词连词to/at/in/on/above等

    选项的秘密：

    • 动词：根据前后搭配
    • 名词：找同类
    • adj/adv：感情色彩：正/负/中性
    • 介词连词：根据逻辑关系。to/at/in/on/above
